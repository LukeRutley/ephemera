# Ephemera

Ephemera is fluid software: an LLM turns intent into fast, task-specific browser interfaces instead of forcing everything through a chat box.

It treats structured data as a first-class advantage for software and persistence, while still mixing in unstructured context when that is the better fit. The system continuously reshapes its database schema around real usage, renders rich custom HTML with very low latency, and can create interfaces that exist for one moment or become familiar saved pages for repeat work. Those saved surfaces are cleaned up and consolidated over time too.

The result is software that adapts to the user instead of asking the user to adapt to the software: from a simple one-off question, to a rich interactive workflow, to something that gradually becomes a fully custom replacement for traditional software.

## Run

```bash
pip install -r requirements.txt
python app.py
```
